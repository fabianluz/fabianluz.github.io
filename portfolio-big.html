IoT Fundamentals: Big Data & Analytics

CISCO certification

Las arquitecturas y los procesos del pasado dejan lugar a aplicaciones más modernas en tiempo real, gracias a los sistemas de IoT (Internet de las Cosas) y los datos que recopilan.

This certification taught me how to use Python to create code that reads sensor data and stores it in a SQL database. To query, clean, manipulate and integrate datasets, and to learn the fundamentals of big data platforms such as Hadoop.

Cursé esta certificación en 2018 como parte de las becas Digitaliza de CISCO.
